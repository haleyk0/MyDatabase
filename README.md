# MyDatabase
CSCE 4350.401 "Build a Database" project
by: Haley Kang (hjk0056)


This simple database is designed to use SET and GET commands. 

Compile/Run:
1. Navigate to directory.
2. Make sure the database file is within the directory and is titled "data.db".
3. To compile type:
   g++ main.cpp
4. To run compiled program type:
   .\a.exe

   As per the instructions, the program requires a command in this form:
   SET <key> <value>
   GET <key>
   EXIT

note: The program will continuously allow for input until it is a valid command.
